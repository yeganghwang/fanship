# `celeb` 컴포넌트

이 디렉토리에는 셀럽 정보 조회 및 관리와 관련된 React 컴포넌트들이 포함되어 있습니다.

## `CelebDetail.js` (예정)

- **설명**: 특정 셀럽의 상세 정보를 표시하는 컴포넌트입니다. `api.md`의 2.4 셀럽 상세 조회 명세에 따라 셀럽 정보를 불러와 보여줄 예정입니다.
- **주요 기능**:
    - `getCelebDetail` API를 호출하여 셀럽 상세 정보를 가져옵니다.
    - 셀럽의 닉네임, 유형, 소속 회사, 인스타그램 URL, 프로필 이미지 URL, 생년월일 등을 표시합니다.
