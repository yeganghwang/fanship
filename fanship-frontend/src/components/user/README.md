# `user` 컴포넌트

이 디렉토리에는 사용자 프로필 조회 및 수정과 관련된 React 컴포넌트들이 포함되어 있습니다.

## `UserProfileDisplay.js`

- **설명**: 사용자 프로필 정보를 표시하는 UI 컴포넌트입니다. `api.md`의 1.4 사용자 프로필 조회 명세에 따라 닉네임, 사용자 유형, 프로필 이미지 URL, 인스타그램 URL 등을 보여줍니다.
- **주요 기능**:
    - `userProfile` 객체를 prop으로 받아 사용자 정보를 렌더링합니다.
    - 프로필 이미지와 인스타그램 URL은 존재할 경우에만 표시됩니다.

## `UserProfileEditForm.js`

- **설명**: 사용자 프로필 정보를 수정할 수 있는 UI 컴포넌트입니다. `api.md`의 1.5 사용자 정보 수정 명세에 따라 닉네임, 비밀번호, 프로필 이미지 URL, 인스타그램 URL 등을 수정할 수 있는 폼을 제공합니다.
- **주요 기능**:
    - `initialData` prop으로 현재 사용자 프로필 정보를 받아 폼 필드를 초기화합니다.
    - 닉네임, 새 비밀번호, 프로필 이미지 URL, 인스타그램 URL 입력 필드를 포함합니다.
    - 비밀번호는 변경 시에만 입력하도록 합니다.
    - 수정 버튼 클릭 시 `updateUserProfile` API를 호출하여 정보를 업데이트합니다.
    - 업데이트 성공 시 `onUpdateSuccess` 콜백 함수를 통해 부모 컴포넌트에 업데이트된 데이터를 전달합니다.
    - 업데이트 실패 시 에러 메시지를 표시합니다.
