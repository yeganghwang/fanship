# `user` 컴포넌트

이 디렉토리에는 사용자 프로필 조회 및 수정과 관련된 React 컴포넌트들이 포함되어 있습니다.

## `UserProfileDisplay.js`

- **설명**: 사용자 프로필 정보를 표시하는 UI 컴포넌트입니다. `api.md`의 1.4 사용자 프로필 조회 명세에 따라 닉네임, 사용자 유형, 프로필 이미지 URL, 인스타그램 URL 등을 보여줍니다.
- **주요 기능**:
    - `userProfile` 객체를 prop으로 받아 사용자 정보를 렌더링합니다.
    - 프로필 이미지와 인스타그램 URL은 존재할 경우에만 표시됩니다.

## `UserProfileEditForm.js`

- **설명**: 사용자 프로필 정보를 수정할 수 있는 UI 컴포넌트입니다. `api.md`의 1.5 사용자 정보 수정 명세에 따라 닉네임, 프로필 이미지 URL, 인스타그램 URL 등을 수정할 수 있는 폼을 제공합니다. 비밀번호 변경 기능은 별도의 컴포넌트(`ChangePasswordForm.js`)로 분리되었습니다.
- **주요 기능**:
    - `initialData` prop으로 현재 사용자 프로필 정보를 받아 폼 필드를 초기화합니다.
    - 닉네임, 프로필 이미지 URL, 인스타그램 URL 입력 필드를 포함합니다.
    - 수정 버튼 클릭 시 `updateUserProfile` API를 호출하여 정보를 업데이트합니다.
    - 업데이트 성공 시 `onUpdateSuccess` 콜백 함수를 통해 부모 컴포넌트에 업데이트된 데이터를 전달합니다.
    - 업데이트 실패 시 에러 메시지를 표시합니다.

## `ChangePasswordForm.js`

- **설명**: 사용자의 비밀번호를 변경하는 UI 컴포넌트입니다. `api.md`의 1.7 비밀번호 변경 명세에 따라 현재 비밀번호와 새 비밀번호, 새 비밀번호 확인을 입력받아 비밀번호 변경 API를 호출합니다. 두 개의 새 비밀번호 필드는 실시간으로 일치 여부 및 최소 8자 길이 유효성 검사를 수행합니다.
- **주요 기능**:
    - 현재 비밀번호(`currentPassword`) 및 새 비밀번호(`newPassword`), 새 비밀번호 확인(`confirmNewPassword`) 입력 필드.
    - 새 비밀번호와 새 비밀번호 확인 필드의 실시간 일치 여부 검사 및 피드백 제공.
    - 새 비밀번호 필드의 최소 8자 길이 유효성 검사 및 피드백 제공.
    - 비밀번호 변경 버튼.
    - 비밀번호 변경 성공 시 메시지를 표시하고 입력 필드를 초기화합니다.
    - 비밀번호 변경 실패 시 에러 메시지를 표시합니다.
