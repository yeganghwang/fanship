# `company` 컴포넌트

이 디렉토리에는 회사 정보 조회 및 관리와 관련된 React 컴포넌트들이 포함되어 있습니다.

## `CompanyList.js`

- **설명**: 회사 목록을 표시하고 필터링 및 페이지네이션 기능을 제공하는 컴포넌트입니다. `api.md`의 2.2 회사 목록 조회 명세에 따라 회사 정보를 불러와 보여줍니다.
- **주요 기능**:
    - `getCompanyList` API를 호출하여 회사 목록을 가져옵니다.
    - 지역(`region`)별 필터링 기능을 제공합니다.
    - 페이지네이션(`page`, `limit`)을 통해 회사 목록을 탐색할 수 있습니다.
    - 로딩 상태 및 오류 메시지를 표시합니다.
