# `favorite` 컴포넌트

이 디렉토리에는 즐겨찾기 관련 React 컴포넌트들이 포함되어 있습니다.

## `FavoriteList.js`

- **설명**: 사용자의 즐겨찾기 목록을 표시하고 삭제 기능을 제공하는 컴포넌트입니다. `api.md`의 3.3 사용자의 즐겨찾기 목록 조회 명세에 따라 즐겨찾기 정보를 불러와 보여줍니다.
- **주요 기능**:
    - `getFavorites` API를 호출하여 즐겨찾기 목록을 가져옵니다.
    - 즐겨찾기 항목(회사 또는 셀럽)을 표시합니다.
    - 즐겨찾기 항목을 삭제하는 기능을 제공합니다.
    - 페이지네이션을 통해 즐겨찾기 목록을 탐색할 수 있습니다.
    - 로딩 상태 및 오류 메시지를 표시합니다.
