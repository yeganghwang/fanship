# `auth` 컴포넌트

이 디렉토리에는 사용자 인증(로그인 및 회원가입)과 관련된 React 컴포넌트들이 포함되어 있습니다.

## `LoginForm.js`

- **설명**: 사용자의 로그인 처리를 담당하는 UI 컴포넌트입니다. 사용자 이름과 비밀번호를 입력받아 로그인 API를 호출합니다.
- **주요 기능**:
    - 사용자 이름(`username`) 및 비밀번호(`password`) 입력 필드.
    - 로그인 버튼.
    - 로그인 성공 시 `onLoginSuccess` 콜백 함수를 통해 부모 컴포넌트에 로그인 정보를 전달합니다.
    - 로그인 실패 시 에러 메시지를 표시합니다.

## `RegisterForm.js`

- **설명**: 사용자의 회원가입 처리를 담당하는 UI 컴포넌트입니다. 필요한 사용자 정보를 입력받아 회원가입 API를 호출합니다. 비밀번호와 비밀번호 확인 필드의 실시간 유효성 검사를 포함하며, 비밀번호의 최소 8자 길이 유효성 검사를 수행합니다.
- **주요 기능**:
    - 사용자 이름(`username`), 비밀번호(`password`), 비밀번호 확인(`confirmPassword`), 이메일(`mail`), 닉네임(`nickname`), 사용자 유형(`position`) 입력 필드.
    - 비밀번호와 비밀번호 확인 필드의 실시간 일치 여부 검사 및 피드백 제공.
    - 비밀번호 필드의 최소 8자 길이 유효성 검사 및 피드백 제공.
    - 회원가입 버튼.
    - 회원가입 성공 시 메시지를 표시합니다.
    - 회원가입 실패 시 에러 메시지를 표시합니다.
